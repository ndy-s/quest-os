# Goal Journey Template — Map of Content

> A reusable system to turn any goal into a structured Obsidian journey with full planning files and daily quests.

---

## How It Works

1. You write your goal using the format in **[[01 Goal Input Format]]**
2. You paste the prompt from **[[02 Master Prompt]]** into Claude, with your goal filled in
3. Claude generates an entire journey folder: MOC, context, strategy, timeline, tracker, and daily quest system
4. From that point, you run daily quests the same way every day

---

## What Gets Generated

| File                          | What it contains                                                              |
| ----------------------------- | ----------------------------------------------------------------------------- |
| `00 MOC.md`                   | Map of content: all files, current phase, how to start daily quests          |
| `01 Context & Why.md`         | Why this goal matters, what the market/opportunity looks like, your strengths |
| `02 Strategy.md`              | Core approach, preparation areas, priorities, resources                       |
| `03 Skill Strategy.md`        | If skill-based: spaced repetition or learning system for the main skill       |
| `04 Timeline & Phases.md`     | Phase breakdown with tasks, week targets, and readiness gates                 |
| `05 Tracking.md`              | Tracker for the core repeatable unit (problems, books, habits, etc.)          |
| `06 Visibility & Branding.md` | How to show progress publicly (LinkedIn, blog, portfolio, etc.)               |
| `07 Target List.md`           | Specific targets: jobs, clients, publishers, programs, whatever applies      |
| `08 Budget & Resources.md`    | Time, money, and tool investment needed                                       |
| `09 Key Decisions.md`         | Settled decisions so you don't revisit them every week                        |
| `Daily Quest Context.md`      | Context file used by Claude to generate daily quests                          |

## Daily Quest Loop (once your journey is generated)

1. Open Claude
2. Say **"daily quest"** — Claude reads your tracker + context file, generates today's quests
3. Do the quests, fill Session Notes
4. Say **"submit quest"** — Claude evaluates and updates your tracker

That's the full loop.
